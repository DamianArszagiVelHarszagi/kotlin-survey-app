# Movies & Cars SURVEY ✨
Very simple survey, you can choose from 2 surveys: Movies or Cars
What type of fan are you?


## Up & running (optional) 🏃‍➡️
Run main.kt (groene play-knop)
In de terminal volg je stappen:
Kies een survey (typ exact de titel zoals getoond)
Vul je naam in
Beantwoord elke vraag door een nummer te kiezen (1, 2, 3, ...)
